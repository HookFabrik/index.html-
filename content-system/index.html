Das Prinzip

Was eine Prompt-Chain wirklich ist.

Ein normales Prompt-Pack gibt dir 50 einzelne Sätze. Jeder startet bei null. Eine Chain verbindet die Schritte, das Ergebnis von Prompt 1 wird zum Input für Prompt 2. Genau da liegt der Unterschied.

Das Prinzip

Was eine Prompt-Chain wirklich ist.

Ein normales Prompt-Pack gibt dir 50 einzelne Sätze — jeder startet bei null. Eine Chain verbindet die Schritte: Das Ergebnis von Prompt 1 wird zum Input für Prompt 2. Genau das ist der Unterschied.

Sieh es selbst

So sieht ein Prompt — und sein Ergebnis — aus.

Jeder Prompt liefert nicht nur Text, sondern nachvollziehbare Logik. Hier der Contrarian-Hook-Generator: Input links, das Ergebnis, das Claude ausgibt, rechts.

// Input · der Prompt
1.3 – Contrarian-Hook-Generator
# Rolle Du bist ein Content-Stratege für kontroverse, aber fundierte Standpunkte. Nische: [NICHE] Zielgruppe: [ZIELGRUPPE] Verbreitete Meinung: [VERBREITETE_MEINUNG] Erstelle 6 Contrarian-Hooks, die dieser Meinung widersprechen. Bedingungen: › jeder Hook durch ein Argument stützbar › "überraschend, aber plausibel" — nicht "falsch, aber laut" › variiere die Schärfe: 2 moderat, 2 deutlich, 2 zugespitzt Format: Hook / Kernargument / Schärfegrad

„Cardio macht dich nicht schlank. Es hält dich nur müde."

Kernargument: Krafttraining erhöht den Grundumsatz nachhaltiger als reines Cardio — der Hook ist provokant, aber im Video sauber belegbar.
